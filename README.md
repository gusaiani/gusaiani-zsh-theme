# gusaiani Oh My Zsh Theme

### Install
Save `gusaiani.zsh-theme` at `~/.oh-my-zsh/custom/themes`

Set `ZSH_THEME` to `gusaiani` in your `~/.zshrc`, before sourcing Oh My Zsh: `ZSH_THEME=gusaiani`
